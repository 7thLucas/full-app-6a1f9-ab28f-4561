# Cafe Dash

## What It Is
A casual simulation game where players take on the role of a cafe owner/waitress running a busy cafe. Players take orders, prepare food and drinks, and deliver them to the correct tables — all against the clock.

## Core Experience
The rush and satisfaction of clearing every table while staying on top of a packed, busy cafe. Fast-paced, rewarding, pressure-and-payoff gameplay loop.

## Target Audience
Casual mobile gamers, kids, and fans of time-management games like Diner Dash who enjoy juggling multiple tasks under pressure.

## Core Gameplay Loop (P0)
Taking orders from customers and delivering food/drinks to the correct table — the mechanic everything else builds on.

## Game Name
**Cafe Dash**

## Genre
Casual / Simulation / Time-Management

## Build Status
✅ MVP shipped. Core loop live: customer arrival with patience timers → order taking → prep stations (Drinks Bar, Kitchen, Desserts) → delivery → table clear → score.
✅ Difficulty scaling system live — 10-level config curve, patience 45s→20s, arrivals 8s→3s, customer cap 3→6. Visible patience bar above each customer (green→yellow→red).
